Tizen Studio

Tizen Studio 3.2 is now launched!      Download

Tizen Studio is an official integrated development environment (IDE) for developing Tizen apps. This environment is built over Eclipse, it presents a unified environment where you can write, edit, debug, build, and publish your app. Tizen Studio runs on Windows®, Ubuntu, and MacOSX. For more information on the latest release, see the [Download](https://developer.tizen.org/development/tizen-studio/download) page. Tizen Studio primarily consists of: - set of fast and feature-rich emulators for devices based on Tizen, for example: Smartphone, TV emulator, Smart watch and much more - extensive tool chains to capture performance, usability, version compatibility - code templates, samples to help you get started - components to support new platforms - system applications and drivers - utilities for compiling applications to Tizen RT (Tizen RTOS kernel variant) Visit the following pages to explore more about Tizen Studio:
