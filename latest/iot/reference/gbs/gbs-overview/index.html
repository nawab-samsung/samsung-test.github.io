GBS (Git Build System)

GBS (git-build-system) is a command line tool that supports Tizen package development. It is used to generate tarballs based on Git repositories, to do local test buildings, and to submit code to OBS (Tizenâ€™s main build service).

Before going into further GBS details, make sure you have set up the development environment and learned how to install and upgrade tools.

Afterwards, become familiar with GBS by reading the following instructions:

  • GBS Configuration describes how to modify the GBS configuration.
  • GBS Reference describes, in more detail, how to use GBS.
  • GBS Frequently Asked Questions describes frequently asked questions.

The source code is tracked in the https://github.com/01org/gbs repository.
